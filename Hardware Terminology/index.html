Input Device

Input Devices turn phsical events into ones and zeros. They allow users to interact and control the computer. Some examples of input devices would include Mouse, Keyboard, Touchscreen, Gyroscope, and Microphone.

Output Device

Output Devices turn ones and zeros into a physical event. The opposite of the Input Device. Some examples of Output Devices are Screens or Moniters, speakers, headphones, printers, and Haptic feedback.

Central Processing Unit(CPU)

The Central Processing Unit or CPU is the brians of the computer. It preforms arithmetic, controlling, logic and input/output operations. The CPU can be divided into two parts The ALU and the control unit.

Arithmetic Logic Unit(ALU)

The Arithmetic Logic Unit or ALU does the bulk of calculations and logic operations. It is one of the two main components of the CPU.

Control Unit

The Control Unit or CU is the second main component in a CPU. It directs the computers memory, ALU, and input/outputs on how to respond to commands that have been sent to the processor. It is the boss of the computer.

Random Access Memory

The Ramdom Access Memory or RAM contains running programs and data the user is currently working with. It is fast and uses logic gates, but is temporary meaning if the computer looses power the data is lost.

Optical Memory

Optical Memory is one of the three long term memory storage devices. It uses lasers to read and write data becuase of this it is slow to write. An Example of opical memory is a DVD.

Magnetic Memory

Magnetic Memory is the second of the three long term memory storage devices. It uses polarized magnetic nodes to repersent ones and zeros. It is an old, slow, but reliable and easy too write to option for storage.

Flash Memory

Flash Memory uses quantum tunnling to store ones and zeros. It is fast, like super fast. It does wear out over a long time, around 10 years. Some examples would be memory sticks or SD hardrives.

Logic Gates

Logic Gates are the fundamental building blocks of all processors. They are made up of transistors and other electronic components such as capacitors and resistors and they simulate Boolian logic (such as and, or, not,) with electricity.
